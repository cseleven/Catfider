# Catfider-Front

## Install

- `node.js`

## npm install

```bash
npm install @headlessui/react@latest
npm i @heroicons/react
npm install @supabase/supabase-js
```

## Learn

- https://learnnextjs.com/next-js-tutorial-getting-started/#updating-indexjs-fast-refresh

## run 1st

```bash
cd frontend
npm install
npm run dev
```

## run

```bash
cd frontend
npm run dev
```
